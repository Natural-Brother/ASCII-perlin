#USAGE:
- "size?" size of the grid (eg:8)
- "debug?" 0 == false and 1 == true, will display gradient vectors
- "width?" number of row (eg:100)
- "height?" number of line (eg:50)
- "range?" default one hundred, how many character of a predetermined ascii shade will be used. (eg:100)
- "octaves?" how many noises will be display ontop of eachother, increase detail (eg:8)
